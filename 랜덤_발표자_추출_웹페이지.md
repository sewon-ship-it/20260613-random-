# 랜덤 발표자 추출 웹페이지

## 개요

수업시간에 발표자 랜덤 뽑아서 보여주는 웹페이지

---

## 목표 및 대상

초등학교 5학년

---

## 핵심 기능

- 몰래 교사가 순서 마음대로 정하는 기능
- 발표자를 랜덤으로 뽑는 기능
  - 추출 인원 선택 기능
- 학생 명단 추가하는 기능
  - 한 명 또는 여러 명을 일괄로 추출할 수 있는 기능
